Набор библиотек для Android

24 декабря 2014 Автор: Роман Татауров

Здесь приведены библиотеки, которые я использую для своих Android поделок. Годы берут своё — в голове уже всё не помещается поэтому скину буфер на эту страницу.

ORM

  • GreenDAO

Минимальные накладные расходы
Используется генерация кода — т. е. пишем отдельное приложение, в котором описываем схему, и которое при запуске генерирует классы для доступа к данным.

Аннотации

  • androidannotations

Сильно облегчает написание кода — многие языковые конструкции заменяются довольно простыми аннотациями.

Внешний вид

HoloEverywhere

Библиотека была актуальна в времена перехода на 4-ю версию Android и позволяла использовать элементы интерфейса 4-й версии (Holo) в более ранних версиях.

ActionBarScherlock

Библиотека позволяла использовать новый ActionBar их 4-й версии Android (Holo) в более ранних версиях.
Те-же грабли — после массового перехода на 4-й андроид библиотека потеряла свою значимость.
